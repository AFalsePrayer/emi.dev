Shenzhen Solitaire GBC /git/emi/shenzhen-solitaire-gbc

An implementation of the solitaire game in Shenzhen I/O by Zachtronics written in C for the Gameboy Color.

A Little About Me

I've been a software engineer since mid 2012. In that time I've written C on the Gameboy Color, leveraged HTML/CSS/JS on the web, and become a master of the Java Virtual Machine.

Many of my projects require an understanding and application of runtime bytecode transformation in Java. If you're looking for me, make sure to double check your classes.

I currently use KDE Neon for my desktop and VSCodium as my IDE.
